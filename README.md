# ComicConverter

A multi-plataform .NET library written in Net standard 2.0 for comic file formating.

## Suported Formats

These are the format that we current support. More in the way!

| Format | Extract Images |  Create Comic  |
| :----: | :------------: | :------------: |
|  CBR   |       ✔️      |       ❌       |
|  CBZ   |       ✔️      |       ✔️       |
|  CBT   |       ✔️      |       ✔️       |
|  CB7   |       ✔️      |       ❌       |
|  PDF   |      ❌       |❌ (Coming soon)|
